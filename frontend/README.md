## 🚀 Overview

This repository hosts a comprehensive personal developer portfolio ecosystem, providing a seamless bridge between my skills, professional experiences, and potential clients. Built with a controlled, pristine design philosophy, the platform handles everything from complex 3D rendering to smooth animated reveals, engineered with absolute precision.

## 💡 Key Features

- ⚡ **Instant Interactivity**: High-performance, non-blocking 3D WebGL scene that responds smoothly to mouse and scroll events.
- 📋 **Professional Standard Components**: Custom HTML, highly accessible UI elements built with `shadcn-ui` and Tailwind CSS.
- 📍 **Precise Mapping & Animations**: Smooth GSAP + Framer Motion-backed scrolling events and staggered reveals.
- ♦️ **Premium UI/UX**: Modern dark mode aesthetic featuring glass-morphism, sophisticated layouts, and strong typography.
- 🚀 **React Performance**: Optimized React loading logic and extremely fast bundle sizes powered by Vite's SWC compiler.

## 📂 Project Structure

```bash
portfolio-frontend/
├── public/              # Static assets and 3D models (including banner)
├── src/
│   ├── components/      # UI components & Sections (Hero, About, Projects)
│   │   ├── three/       # 3D WebGL scenes and custom rendering logic
│   │   └── ui/          # Reusable shadcn/ui elements (Toasts, Buttons)
│   ├── hooks/           # Custom React hooks
│   ├── image/           # Optimized raster graphics and previews
│   ├── lib/             # Utility functions (tailwind-merge, clsx)
│   ├── pages/           # Route logic (Index, NotFound)
│   └── App.jsx          # Main App entry with global providers
└── package.json         # Project dependencies & scripts
```

## 🔧 Installation & Setup

### 1. Repository Setup

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate into the frontend directory
cd frontend
```

### 2. Frontend Configuration

```bash
# Install NPM dependencies
npm install

# Start the Vite development server
npm run dev
```
